# U&ME Cafe Website

U&ME is a modern and aesthetic cafe website built using HTML, CSS, and JavaScript.  
It features a soft baby pink theme with theme switching options, category-based menu filtering, and a customer review section.

## Features

- Responsive layout with a clean modern design  
- Theme switcher with 3 color options:
  - Baby Pink (default)
  - Dark Pink
  - Coffee Brown  
- Category tabs for menu filtering:
  - All
  - Coffee
  - Desserts
  - Snacks  
- Customer reviews section  
- Smooth navigation and button interactions  
- Simple contact form with submission alert  

## Technologies Used

- HTML
- CSS
- JavaScript
